This is web service takes legal documents from EURLex, compares them and outputs the differences. The result will consist of Changes, which when opened display all their modifications. Each Change has Meta-Data, a set of information about which amending act produced the modification of this change and when. Each modification names an assumed type of modification (based on the diff): (Inserted) Addition, (Inserted) Deletion or (Inserted) Replacement. This may vary from the actual type of modification. Furthermore, the rough position this modification is in, so with the provided links on the top, each modification can be double-checked. The new content is output with highlighted entities. The categories are explained below. Under "the corresponding passage", the context of the modification as well as the old (red) and new (green) text. The web service provides the possibility to either enter both links (old and new) or only the old document by either a link or the CELEX number from EURLex.

Use:

Two links

One (old) Link

CELEX Number (of old document)


The highlighted and labeled entities are:

LAW: Legal References, ORG: Organizations, GPE: Countries, NORP: Nationalities, LOC: Locations, DATE, QUANTITY, MONEY, LANGUAGE.
